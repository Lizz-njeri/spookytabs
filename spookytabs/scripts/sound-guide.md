# Sound Creation Guide

## Quick Start: Free Sound Resources

### Best Free Sound Libraries

1. **Freesound.org** (Best for variety)
   - Search terms: "whisper", "creak", "ambient horror", "forest night"
   - Filter by: Creative Commons 0 (public domain)
   - Download as MP3 or convert WAV to MP3

2. **Zapsplat.com** (High quality)
   - Free with attribution
   - Great horror/spooky category
   - Pre-organized by theme

3. **Pixabay Audio** (No attribution required)
   - Smaller selection but completely free
   - Good for ambient loops

4. **BBC Sound Effects** (Public domain)
   - Professional quality
   - Search "horror", "night", "creaking"

## DIY: Create Your Own Sounds

### Using Audacity (Free)

1. **Install Audacity** (audacityteam.org)

2. **Create Ambient Loop:**
   - Generate → Noise (pink noise for wind)
   - Effect → Low Pass Filter (make it deeper)
   - Effect → Reverb (add space)
   - Effect → Fade In/Out (for seamless loop)
   - Export as MP3

3. **Create Whisper:**
   - Record yourself whispering
   - Effect → Reverb (add eeriness)
   - Effect → Change Pitch (lower slightly)
   - Trim to 1-2 seconds

4. **Create Creak:**
   - Record door or chair creaking
   - Effect → Amplify
   - Effect → Reverb
   - Trim to 1-2 seconds

### Quick Tips

- **Volume:** Keep sounds subtle (not startling)
- **Format:** MP3, 128-192 kbps
- **Length:** Ambient 30-60s, Effects 1-3s
- **Looping:** Use fade in/out for seamless loops
- **Testing:** Test at different volumes

## Legal Notes

- Use only royalty-free or Creative Commons 0 sounds
- Check license requirements (some need attribution)
- Don't use copyrighted movie/game sounds
- When in doubt, create your own!

## Testing Your Sounds

1. Place MP3 files in correct folders
2. Reload extension
3. Open new tab
4. Check browser console for errors
5. Adjust volumes in settings
