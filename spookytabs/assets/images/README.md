# Extension Icons

This directory contains the extension icons in various sizes required by Chrome Web Store.

## Icon Sizes
- `icon16.png` - 16x16px (toolbar icon)
- `icon32.png` - 32x32px (toolbar icon @2x)
- `icon48.png` - 48x48px (extension management)
- `icon128.png` - 128x128px (Chrome Web Store)

## Generating New Icons

**Easy Method:** Open `scripts/generate-icons.html` in your browser and click "Download All Icons".

The generated icons feature:
- A ghostly figure with wavy bottom edge
- Dark purple gradient background (#1a0f2e to #2d1b4e)
- Purple glow effect (#8b4f9e)
- Professional vector-style design

## Icon Requirements

- PNG format with transparency
- Square dimensions
- Clear and recognizable at small sizes
- Dark theme friendly
